The Arduino Hub

Site Plan

Site Name

The Arduino Hub

This name positions the site as a central resource and gathering place for beginners in the Arduino community.

Site Purpose

The Arduino Hub is an online resource dedicated to helping beginners learn and explore the exciting world of Arduino. We provide comprehensive and easy-to-follow tutorials on a wide range of Arduino projects, from simple LED blinking to more complex home automation systems. Our goal is to empower aspiring makers with the knowledge and skills they need to bring their creative ideas to life. We also offer valuable information on essential components and provide a platform for the community to share project ideas and collaborate.

Style Guide

Color Schema

Primary Color: #007BFF (A vibrant shade of blue)

  • Header and footer
  • Primary call-to-action buttons
  • Headings for key sections
  • Links (hover state)

Secondary Color: #28A745 (A cheerful shade of green)

  • Success messages and notifications
  • Background color for success blocks
  • Accents in diagrams and illustrations
  • Nav (hover state)

Neutral Color 1: #FFFFFF (Pure white)

  • Background color for most of the website
  • Text color for headings
  • Secondary text color for use over main colors

Neutral Color 2: #333333 (Dark gray)

  • Text color for body copy and smaller text elements
  • Headings in less prominent sections
  • Border color for elements

Typography

Primary Font: Open Sans

  • Body text
  • Code blocks

Secondary Font: Poppins

  • Headings (H1-H3)
  • Button labels
